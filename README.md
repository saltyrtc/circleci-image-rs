Docker image for CircleCI, based on the official `rust` image.

## Additions

**SaltyRTC**

- saltyrtc-server-python
- libsodium

**C tools**

- cmake
- make
- meson
- ninja

**Rust tools**

- cargo-audit

**Code analysis**

- valgrind
- splint
